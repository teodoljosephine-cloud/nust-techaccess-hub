About This App

N

NUST TechAccess Hub

An offline-first Progressive Web App for NUST students — lightweight, device-friendly, and built to work even without internet.

  • 📁
    Real File Upload & DownloadUpload PDFs, PPTXs, videos & more. Files are saved directly to your device using IndexedDB — no server needed.
  • 👁
    In-App PreviewPreview PDFs, images and videos right inside the app before downloading.
  • 📡
    Offline-First (PWA)Service Worker caches the app shell. Works without Wi-Fi after the first load.
  • ⚡
    Data Saver ModeStrips all styling to raw text — uses only a few kilobytes of data.
  • 📶
    Bluetooth / ShareIt ReadyDownload any file to your device, then share peer-to-peer — no internet needed.

Prototype v2.0 · Built by a NUST student for IT presentation.
Stack: HTML5 · CSS3 · Vanilla JS · IndexedDB · Service Worker
